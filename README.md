# BI Templates Lab

This repository contains a collection of Business Intelligence templates:
- Power BI

- Google Sheets
- Notion

A professional portfolio template for freelancers and BI developers.

## Folder Structure

## How to Use
1. Clone this repo
2. Open PBIX template
3. Customize your data
4. Publish portfolio
